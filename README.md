# 📚 Open Source Book Finder App

A simple and user-friendly web application to search books using the **Open Library API**.  
Search books by title, view key details, and explore a world of knowledge!  
This project is built using **React.js** and is completely open-source.  

---

## ✨ Features

✅ Search books by title  
✅ Fetches real book data from **Open Library API**  
✅ Displays book cover, title, author, and publish year  
✅ Responsive UI & clean modern design  
✅ Loading spinner & no-result message  
✅ SEO-friendly content and meta tags
